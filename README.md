# Restaurant List

紀錄喜歡的餐廳名單，可以瀏覽及查看詳細餐廳資訊

## 功能

- 查看所有餐廳
- 依選擇分類排序餐廳
- 查看餐廳詳細資訊
- 可依店名或餐廳類別收尋想要的餐廳
- 連結餐廳地址到 Google Map
- 加入 MySQL 資料庫存儲餐廳資訊
- 可收藏新餐廳
- 可編輯餐廳資訊
- 表單輸入錯誤提示

## 開始使用

1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，在終端機輸入：

```
npm install
```

4. 安裝完畢後，繼續輸入：

```
npm run start
```

5. 當終端機出現以下訊息，表示順利運行，打開瀏覽器輸入網址

```
App is running on http://localhost:3000
```

6. 停止使用 ctrl + c
